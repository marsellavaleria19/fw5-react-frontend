## 📌 How To Install?

- Clone This Repo

```
git clone https://github.com/marsellavaleria19/rvrental-frontend.git
```

- Go To Folder Repo

```
cd fw5-react-frontend
```

- Install Module

```
npm install
```

- Type ``` npm run start ``` To Start The Project
